# gis_utils

GIS/CAD utility library. Installed as editable in conda env `gis`:
```
pip install -e ~/dev/Gunther-Schulz/gis_utils
```

## Plugin

The `gis-utils` Claude Code plugin provides:
- **MCP tools** for live API discovery — use `mcp__gis-utils__catalog`, `mcp__gis-utils__list_recipes`, `mcp__gis-utils__list_templates`, `mcp__gis-utils__get_function_help` instead of reading this file
- **Skills** auto-loaded per turn: `gis-safety` (CRS, defaults, conventions), `geometry-workflow` (analyze before coding), `workflow-authoring` (workflow.yaml, templates, recipes)
